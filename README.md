# Sistema de Gestão de Evento

## Descrição
Programa em Python que integra conceitos de:

- Estruturas condicionais
- Estruturas de repetição
- Listas
- Dicionários
- GitHub para versionamento

## Funcionalidades
1. Verificação de idade para entrada no evento.
2. Contagem de números pares de 1 a 100 (for e while).
3. Cadastro de participantes (lista).
4. Cadastro de produtos disponíveis no evento (dicionário).

## Estrutura do projeto
- `main.py`: Programa principal.
- `participantes.py`: Funções para gerenciar lista de participantes.
- `produtos.py`: Funções para gerenciar produtos.
- `utils.py`: Funções auxiliares (verificação de idade e contadores).
