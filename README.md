1. ESTRUTURAS CONDICIONAIS — (1,0 pt)
------------------------------------------------------------

Descrição:
Programa que simula um sistema de verificação de idade para entrada em eventos.
O usuário informa a idade, e o sistema informa se a entrada é permitida, proibida ou restrita.

Critérios atendidos:
- Uso correto de if, elif, else
- Entrada e saída de dados
- Clareza e comentários

Como executar:
python condicional/condicional.py

Exemplo:
Entrada:
17
Saída:
Entrada proibida para menores de 18 anos.

2. ESTRUTURAS DE REPETIÇÃO — (1,0 pt)
------------------------------------------------------------

Descrição:
Criar dois contadores exibindo apenas números pares de 1 a 100:
- Um utilizando FOR
- Outro utilizando WHILE

Critérios atendidos:
- Laços implementados corretamente
- Uso de operadores lógicos
- Código organizado

Como executar:
python repeticao/repeticao.py

Exemplo de saída:
Contador com FOR:
2 4 6 ... 100

Contador com WHILE:
2 4 6 ... 100


------------------------------------------------------------
3. LISTAS — (1,0 pt)
------------------------------------------------------------

Descrição:
Programa que recebe nomes de alunos, armazena em uma lista e exibe ao final.

Critérios atendidos:
- Uso adequado de listas
- Entrada dinâmica
- Impressão correta dos elementos

Como executar:
python listas/listas.py

Exemplo:
Entrada:
Digite um nome: Ana
Digite outro nome? (s/n) s
Digite um nome: Pedro
Digite outro nome? (s/n) n

Saída:
Alunos cadastrados:
Ana
Pedro


------------------------------------------------------------
4. DICIONÁRIOS — (1,0 pt)
------------------------------------------------------------

Descrição:
Sistema simples de cadastro de produtos com nome e preço usando dicionários.

Critérios atendidos:
- Uso correto de dict()
- Inserção e recuperação de dados
- Estrutura clara e funcional

Como executar:
python dicionarios/dicionarios.py

Exemplo:
Entrada:
Produto: Sabonete
Preço: 4.50
Adicionar mais? (s/n): s
Produto: Shampoo
Preço: 12.90
Adicionar mais? (s/n): n

Saída:
Produtos cadastrados:
Sabonete - R$4.50
Shampoo - R$12.90
